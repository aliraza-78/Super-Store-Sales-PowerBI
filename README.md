# Super Store Sales Dashboard | Power BI

## Overview
This project is an interactive Power BI dashboard built using Super Store sales data.  
It analyzes sales performance, profit trends, and regional insights, and includes a short-term sales forecast.

The goal of this project is to practice business-focused data analysis and dashboard design using Power BI.

---

## Key Features
- KPI cards for **Sales**, **Profit**, **Quantity**, and **Shipping Days**
- Year-over-Year (YoY) analysis for sales and profit
- Sales breakdown by **Region**, **Category**, **Sub-Category**, **Segment**, and **State**
- Interactive filters for regional analysis
- **15-day sales forecast** using time series forecasting

---

## Dashboard Pages
1. **Sales Overview Dashboard**
   - Overall performance KPIs
   - Monthly sales and profit trends
   - Category and region-wise analysis

2. **Sales Forecast Dashboard**
   - Historical sales trend
   - 15-day future sales forecast
   - State-wise sales comparison

---

## Tools & Technologies
- Power BI Desktop
- Power Query
- DAX (basic measures)
- Time series forecasting (Power BI built-in)
